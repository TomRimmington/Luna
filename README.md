
  # LUNA AI Interface

  This is a code bundle for the LUNA AI Interface. The original project is available at https://www.figma.com/design/0fx9PvO5rbXskU869X9Is2/Design-LUNA-AI-Interface.

  ## Running the code

  Run `npm i` to install the dependencies.

  Run `npm run dev` to start the development server.
  
Note: You will need to create your own env file with your own API keys to integrate and run the system. For the project policy, we have to keep our API keys private but you have to generate your own.
